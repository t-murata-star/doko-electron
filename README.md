# 開発環境

## アプリケーション起動
`$ npm run react:start`

## Electron起動@windows
`$ npm run electron:start:windows`

## Electron起動@macOS
`$ npm run electron:start:mac`

# デプロイ

## アプリケーションのビルド
`$ npm run react:build`

## Electronのビルド@windows
`$ npm run electron:build:windows`

## Electronのビルド@macOS
`$ npm run electron:build:mac`
